# Sistema de login - Lista de Usuários

## Depêndencias de Software
- Vue3 instalado
- Node.js: Versão 20.0 ou superior
- MongoDB rodando em uma instância local
- gerenciador de pacotes (npm, yarn)

## Setup do projeto
```
git clone https://github.com/gabrii3lmao/Login-plain-page.git
```

## Variávels de ambiente
```
# String de conexão do MongoDB 
MONGO_URI=mongodb://localhost:27017/nome_do_seu_banco

# Chave secreta para assinar o JWT
JWT_SECRET=sua_chave_secreta_aqui
```
